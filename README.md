# Spell Failure
This extension rolls arcane failure chance when the cast button of the spell on the actions tab is clicked.

# Features
* Roll arcane spell failure chance [when appropriate](https://www.fantasygrounds.com/forums/showthread.php?48977-Advanced-3-5e-and-Pathfinder-effects&p=528377&viewfull=1#post528377).
* Alternately, based on option toggle, prompt user to roll via a chat message which includes the spell failure chance.

# Compatibility and Configuration
This extension has been tested with [FantasyGrounds Classic](https://www.fantasygrounds.com/home/FantasyGroundsClassic.php) 3.3.11 and [FantasyGrounds Unity](https://www.fantasygrounds.com/home/FantasyGroundsUnity.php) 4.0.0 (2020-07-16).
